# How to run

> Backend was made by Teacher (idk his github account name to add him in Contributors)

---

Firstly clone the repository `git clone https://github.com/nezuss/Frontend_Booking.git`

Go to the backend directory and run this commands in 2 diffrent CMD/PowerShell/Terminal(Linux/MacOS):
 - (Window 0) Install npm packages `npm i`
 - (Window 0) Run frontend `npm run dev`

Go to the frontend directory and run this commands in 2 diffrent CMD/PowerShell/Terminal(Linux/MacOS):
 - (Window 0) Install npm packages `npm i`
 - (Window 0) Run frontend `npm run dev`

Go to `http://localhost:5173/`
